# Zen Mode for X (Twitter)

A Chrome extension that hides engagement metrics on Twitter/X to reduce doomscrolling.

## Features

- Hide like counts
- Hide retweet counts  
- Hide reply counts
- Hide view counts
- Customizable via popup settings
